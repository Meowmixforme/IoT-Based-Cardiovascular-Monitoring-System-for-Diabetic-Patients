On a Windows Machine:

1) Launch a command prompt window (CMD) after having Python installed on the local machine

2) Paste pip install -r requirements.txt in a CMD to install the required packages.

3) Extract the folder from the Zip file and navigate to it in the CMD e.g. (cd %USERPROFILE%\Desktop\your_folder_name) . Paste 'streamlit run Diabetic-Heart.py' into the command window to run the app.
